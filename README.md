# 🧊 冰箱遗书

> **别让番茄死得不明不白** 🍅

一个帮助记不住冰箱里有什么的人管理食材的工具。用黑色幽默的方式，把过期食材拟人化，让管理食材变得有趣而不沉重。

## ✨ 已实现功能

### 1. 食材卡片流

- ✅ 所有食材以卡片形式展示
- ✅ 按剩余保质期天数排序（快过期的在最上面）
- ✅ 根据剩余天数显示不同颜色状态：
  - 🟢 ≥ 7 天：保鲜绿
  - 🟡 4-6 天：黄绿色渐变
  - 🟠 2-3 天：橙黄色预警 + "⚠️ 尽快吃"
  - 🔴 1 天：橙红色 + "⏰ 明天到期"
  - 🔥 0 天：红色闪烁 5 次 + "🔥 今天吃"
- ✅ 点击删除按钮移除食材
- ✅ 卡片滑入动画效果

### 2. 快速录入

- ✅ 支持智能文本解析
- ✅ 自动识别食材名称和数量
- ✅ 示例：
  - "鸡蛋一盒" → 食材：鸡蛋，数量：一盒
  - "番茄 3 个" → 食材：番茄，数量：3 个
  - "菠菜、牛奶" → 分别创建两条记录
- ✅ 内置食材保质期数据库（自动匹配预估保质期）
- ✅ 支持回车快速添加

### 3. "下一顿吃什么" 推荐

- ✅ 智能分析当前库存
- ✅ 优先推荐消耗即将过期食材的菜谱
- ✅ 推荐逻辑：
  1. 优先用今天/明天到期的食材
  2. 其次用 2-3 天内的食材
  3. 没有临期食材时随机推荐
- ✅ 显示菜名 + 用到的食材
- ✅ 5 秒后自动隐藏推荐卡片

### 4. 空状态引导

- ✅ 冰箱为空时显示友好提示
- ✅ "📭 冰箱空空如也，去买点菜吧"
- ✅ 禁用推荐按钮防止误操作

### 5. 数据持久化

- ✅ 使用 localStorage 保存食材数据
- ✅ 刷新页面数据不丢失

### 6. 视觉设计

- ✅ 符合需求文档的配色方案
- ✅ 冰箱白 + 番茄红 + 保鲜绿
- ✅ 圆角卡片、柔和阴影
- ✅ 响应式设计，支持移动端
- ✅ 流畅的动画效果

## 📱 使用示例

### 添加食材

在输入框中输入：

```
鸡蛋一盒、番茄 3 个、菠菜一把
```

点击"添加"按钮或按回车，系统会自动解析并创建 3 条记录。

### 查看食材状态

- 绿色卡片：还有至少 7 天，很新鲜
- 橙色卡片：只剩 2-3 天，尽快吃
- 红色卡片：今天必须吃掉！

### 获取菜谱推荐

点击底部的"🍽️ 下一顿吃什么"按钮，系统会根据你的库存推荐菜谱。
